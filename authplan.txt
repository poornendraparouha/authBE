These are common production requirements.

1. Rate Limiting ⭐⭐⭐

Prevent brute-force login attacks.

Install:

npm install express-rate-limit

Example:

const loginLimiter = rateLimit({
  windowMs: 15 * 60 * 1000,
  max: 5,
});

Apply:

router.post("/signin", loginLimiter, userLogin);
2. Helmet ⭐⭐⭐

Security headers.

npm install helmet
import helmet from "helmet";

app.use(helmet());

Adds:

X-Frame-Options
Content-Security-Policy
X-XSS-Protection
3. Input Sanitization ⭐⭐⭐

Prevent NoSQL Injection.

npm install express-mongo-sanitize
app.use(mongoSanitize());

Blocks:

{
  "email": {
    "$gt": ""
  }
}
4. Request Validation ⭐⭐⭐

Currently:

if (!email || !password)

Production:

npm install zod

or

npm install joi

Validate all incoming requests.

5. Centralized Error Handler ⭐⭐⭐

Instead of:

try {
}
catch(error){
}

everywhere.

Create:

middlewares/errorHandler.js

Then:

app.use(errorHandler);

Much cleaner.

↓

↓
